Class
ResultsPrinter This class prints out the EPS, LaTex, and raw data files for a group of random walks
Fields
c Type: CAStatic
       This is the main class. Storing a link to it here gives access to all the data required for printing.

       All the following strings are default file names which are overwritten.

dataFilename = "file.dat" Type: String
       
timeString = "00" Type: String
       
twoPlaces Type: static DecimalFormat
       This is used for printing floating point values to two decimal places.
dir Type: File
       The directory in which to put the output files. Will be created according to the date, hour, and minute of the run.
resultcount Type: static int
       Currently unused. Could count the number of results written.
maxdots = 50 Type: static int
       max size of histogram. Histograms of larger size will be scaled. This stops the eps files using too many pixels.
sf Type: int
       Scale factor for histogram. Must be applied for all results in the group of walks.
